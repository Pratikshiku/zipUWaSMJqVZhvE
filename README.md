## Great Health Elderly Care Apartment Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/35eb8f178fa14aafb8168b450ac39dce/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, caregivers, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Fee Standards
6. Reception Information
7. Room Information
8. Check-in Information
9. Leave Information
10. Reminder Information
11. Extension Information
12. Care Information
13. Check-out Information
14. Announcement Information

**Caregiver Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Fee Standards
6. Reception Information
7. Room Information
8. Check-in Information
9. Leave Information
10. Reminder Information
11. Extension Information
12. Care Information
13. Check-out Information
14. Announcement Information



**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. User Management
4. Caregiver Management
5. Fee Standards Management
6. Reception Information Management
7. Room Type Management
8. Room Information Management
9. Check-in Information Management
10. Leave Information Management
11. Reminder Information Management
12. Extension Information Management
13. Care Information Management
14. Check-out Information Management
15. System Management (announcement information)
